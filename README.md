# 跟着黑马学 AI

这是我的AI学习路线的代码仓库，用来记录从 Python 基础、数据库、数据分析、机器学习到深度学习的学习过程。
仓库里既有课堂代码，也有自己写的小练习、小项目、数据处理脚本和建模实验。

## 学习路线

目前仓库内容大致可以分成几个阶段：

1. Python 编程基础
2. Python 高级语法与小项目
3. Web 自动化与爬虫
4. Flask 服务端基础
5. SQL 数据库基础
6. Pandas、Matplotlib、文本分析等数据分析内容
7. 传统机器学习与集成学习
8. 深度学习、自动微分、神经网络、图像分类和目标检测
9. 数学建模、课程作业、比赛实践与综合项目

## 项目结构

```text
.
├── Python_study/              # Python 基础、高级语法、爬虫、Flask、小游戏等练习
├── 数据库/                    # SQL 基础语句练习，包括 DDL、DML、DQL
├── 数分/                      # 数据分析学习内容，包括 Pandas、Matplotlib、文本预处理等
├── 机器学习/                  # 机器学习与集成学习实验，如随机森林、GBDT、XGBoost
├── deep♂door♂learning/        # 深度学习学习内容，包括自动微分、神经网络、图像项目
└── workworkworkwork/          # 课程作业、数学建模、LDA 主题模型、评论分析等综合实践
```


## 各目录学到了什么

### Python_study

这个目录主要记录 Python 编程学习过程，包含了从基础语法到更高级的语言应用。

截止目前，内容包括：

- Python 基础语法：变量、数据类型、输入输出、格式化字符串
- 流程控制：`if`、`for`、`while`、`while-else`
- 常用数据结构：字符串、列表、元组、字典、集合
- 函数：函数定义、参数、返回值、嵌套调用
- 异常处理：`try-except`、简单文件异常处理
- 文件读写：读取文本、统计内容、写入结果文件
- 面向对象：类、对象、类属性、对象方法
- 小项目练习：学生管理系统、简易计算器、点餐系统、银行系统、猜拳游戏等
- 高级语法：闭包、装饰器、时间统计装饰器、带参数装饰器、requests爬虫实战
- Playwright：浏览器自动化、CSS 定位、角色定位、trace 跟踪、Steam 评论抓取实战
- Flask：简单服务端封装

代表性文件示例：

- `Python_study/Playwright/STEAM游戏评论抓取.py`：使用 Playwright 自动化框架抓取 Steam 游戏评论，实现页面滚动、元素定位、数据提取和 CSV 保存。
- `Python_study/高级编程阶段/网络编程/爬虫实战案例/b站视频爬取.py`：完整的 B 站视频下载器，支持单集和多集视频下载，包含音视频分离下载与合并处理。
- `Python_study/高级编程阶段/网络编程/爬虫基础/requests模块功能.py`：掌握 requests 库核心用法，包括请求头设置、User-Agent 池、URL 参数加密解密、params 传参等反爬技术。
- `Python_study/高级编程阶段/网络编程/爬虫基础/session登录西大.py`：使用 Session 对象维持会话状态，实现需要登录的网站数据采集。
- `Python_study/Flask封装/Flask服务端.py`：基于 Flask 框架构建 Web API 服务，理解路由、请求处理、JSON 响应等后端开发核心概念。
- `Python_study/高级编程阶段/网络编程/爬虫实战案例/豆瓣top榜（get带参）.py`：GET 请求带参数爬取豆瓣电影 Top250 榜单数据，练习分页爬取技巧。


### 数据库

这个目录用于记录 SQL 基础学习内容。

主要文件包括：

- `ddl操作语句.sql`：学习数据库、数据表的创建、修改和删除。
- `dml操作语句.sql`：学习数据的新增、修改、删除。
- `dql操作语句.sql`：学习数据查询，包括条件查询、排序、分组等。

这部分的重点是理解数据库的基本操作方式：先设计表结构，再写入数据，最后通过查询语句获取自己想要的信息。

### 机器学习

这个目录主要用于传统机器学习和集成学习实验。

已覆盖内容包括：

- 随机森林
- AdaBoost
- GBDT
- 网格搜索调参
- Titanic 数据集相关练习
- XGBoost南方电网负载预测实战项目

代表性文件示例：

- `姬器学习/electric_load_predict_project/`：南方电网电力负荷预测完整项目，包含数据预处理、特征工程、可视化分析、XGBoost 模型训练（网格搜索调参）、模型评估和预测全流程，采用模块化设计（src/utils/log/model/data）。
- `姬器学习/KNN/KNN手写数字识别案例.py`：基于 KNN 算法的手写数字识别项目，实现 MNIST 数据集的加载、数据归一化、网格搜索交叉验证调参、模型训练与保存（pickle），支持自定义图片预测。
- `姬器学习/逻辑回归/逻辑回归-电信用户流失预测.py`：电信用户流失预测实战，掌握 One-Hot 编码处理分类变量、特征选择、数据标准化、逻辑回归建模，以及准确率、精确率、召回率、F1 值等多维度模型评估。
- `姬器学习/线性回归/线性回归_梯度下降.py`：从零实现线性回归的梯度下降算法，对比正规方程解法，深入理解损失函数、学习率、迭代优化等核心概念。
- `姬器学习/集成学习/`：集成学习算法全家桶，包括随机森林、AdaBoost、GBDT、XGBoost 在多个数据集上的训练、评估和性能对比，掌握 Bagging 和 Boosting 两大集成思想。

这部分的核心收获是：开始从“入门训练模型”过渡到“设计完整模型训练项目”，理解训练集、测试集、特征、标签、模型训练、模型调参和评估指标。

已覆盖内容包括：

- 自动微分
- 线性回归模拟
- PyTorch 基础
- 激活函数：Sigmoid、Tanh、ReLU、Softmax

- 目标检测项目
- PaddleOCR、YOLOv10 等拓展项目资料

代表性文件示例：

- `deep♂door♂learning/ANN手机价格预测/`：基于人工神经网络的手机价格多分类预测项目，采用模块化设计（src/model/utils），实现数据加载、模型构建、训练循环、评估测试和模型保存的完整深度学习工作流。
- `deep♂door♂learning/CNN_cfar-10图像分类/`：CIFAR-10 图像分类实战，构建自定义 CNN 模型，掌握卷积层、池化层、全连接层的设计，使用 DataLoader 高效加载数据，支持 GPU 加速训练和模型持久化。
- `deep♂door♂learning/梯度下降优化方法/梯度下降优化方法.py`：深入对比四大优化算法（Momentum、AdaGrad、RMSProp、Adam），从理论公式到 PyTorch 实现，理解学习率自适应调整和动量机制如何加速模型收敛。
- `deep♂door♂learning/神经网络搭建/神经网络搭建.py`：PyTorch 神经网络模块化搭建实践，掌握 nn.Module、nn.Sequential、自定义层设计，以及激活函数（ReLU、Sigmoid、Tanh）的选择与应用。
- `deep♂door♂learning/自动微分/自动微分演示.py`：深入理解 PyTorch 自动微分机制（autograd），掌握 requires_grad、backward()、grad 属性，从零实现线性回归训练过程，理解梯度计算与参数更新原理。


这部分的重点是理解深度学习的训练流程：准备数据、定义模型、计算损失、反向传播、更新参数、评估效果。

- 图表输出和 CSV / JSON 结果保存

这个目录的核心意义是练习“项目化思维”：不是只写一个语法点，而是围绕一个问题完成数据收集、清洗、分析、建模和结果展示。
